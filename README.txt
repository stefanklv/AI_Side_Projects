Side projects with useful tools applying AI for practical purposes in business and personal brands. 

For the code to work, the libraries used in the code must be installed by entering "pip install [library]" in the CMD. 

You will also have to enter your OpenAI API key as an environment variable. This can either be done directly on your computer, or by adding the line below early in the code:

os.environ['OPENAI_API_KEY'] = 'Enter API key here'
